## 3. 왜 만들었는지?

평소 영화 리뷰 사이트를 자주 이용하던 팀원이 자기가 보유한 식물도 리뷰하고 재배 난이도 및 생존력 등 정보를 공유해서 타인에게 추천도를 공유하면 좋겠다는 아이디어를 가지고 만들었습니다. <br>
또한 각 식물 데이터의 탄소 흡수량 데이터를 활용하여 개인이 보유한 식물 숫자를 기반하여 환경기여도를 확인 할 수 있게 추가적으로 구현하였습니다.
